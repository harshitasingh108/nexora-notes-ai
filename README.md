# ✨ Features

- 📝 Create, Read, Delete Notes
- 🔐 User Authentication (Login/Signup)
- 🤖 AI Chatbot Integration
- 🔍 Search Notes
- 🌐 Fully Deployed (Frontend + Backend)

## 🛠 Tech Stack

- Frontend: React (Vite + Tailwind)
- Backend: Node.js + Express
- Database: MongoDB
- AI: OpenRouter API
